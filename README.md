# Residential_segregation_2026
Data and code for paper: Residential segregation assessment based on multi-source data and random forest method: A case study of Nanjing
The data compressed file includes POI data, AOI data, housing transaction data, road network data, NDVI data and other basic geospatial information data.
The code compressed file includes the code for classifying compounds using the random forest method and the relevant algorithms for implementing residential isolation analysis.
